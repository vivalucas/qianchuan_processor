# 千川投流视频格式转换

> 一键将竖屏视频批量转换为符合抖音千川/快手等广告平台要求的竖屏格式（竖版视频：宽高比9:16，视频码率≥516kbps，大小≤1000M，720x1280≤分辨率≤1440x2560）

## ✨ 功能特点
- 批量转换处理
- 自动检测视频是否符合平台规范
- 不符合时智能裁剪+转码（保持画质）
- 符合时直接复制（极速处理）
- 无音频视频自动识别与标识
- 无需安装 FFmpeg，开箱即用
